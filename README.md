# 🛍️ Retail Sales Analysis

An end-to-end data analysis project built using Python, SQL, and Matplotlib to uncover sales trends, customer behavior, and product performance insights from retail transaction data.

## 🛠️ Tools Used
* Python
* SQL (SQLite/MySQL)
* Pandas
* Matplotlib
* Jupyter Notebook

## 📊 Project Highlights
* Monthly Revenue Analysis
* Customer Segment Analysis
* Product Category Performance
* Sales Trend Visualization
* Gender-wise Revenue Contribution
* KPI Reporting and Business Insights

## 🧹 Data Preparation
* Cleaned and validated raw sales data
* Removed duplicates and handled missing values
* Standardized data formats and data types
* Queried and aggregated data using SQL
* Performed exploratory data analysis using Pandas

## 🔍 Key Insights
* December sales increased by 25% compared to previous months.
* Electronics contributed approximately 40% of total revenue.
* Female customers accounted for around 60% of overall sales.
* Top-performing products and customer segments were identified through trend analysis.
